# nir_app

Стихи наизусть - приложение для заучивания стихов на Flutter.

## Описание

Приложение для заучивания стихов с использованием распознавания речи (speech-to-text).

## Технологии

- Flutter
- Dart
- speech_to_text

## Установка и запуск

1. Убедитесь, что у вас установлен Flutter SDK
2. Установите зависимости:
   ```bash
   flutter pub get
   ```
3. Запустите приложение:
   ```bash
   flutter run
   ```

## Платформы

- Android
- iOS
- Web
- Linux
- macOS
- Windows
